Projet 1

🛠️ Déploiement industriel & optimisation de scripts

Terminaux embarqués JLT

Le gros morceau de mon stage a été la modernisation du parc de terminaux embarqués (JLT) installés directement sur les chariots élévateurs des préparateurs de commandes. L'objectif ? Remplacer les anciennes machines obsolètes par des terminaux durcis, performants et sécurisés.

💻 Mon défi : dompter le PowerShell

Pour éviter de configurer chaque machine à la main, l'équipe m'a fourni un script d'automatisation. Problème : le script plantait systématiquement au premier démarrage sur la configuration Wi-Fi.

Le problème (asynchronisme) : le script allait plus vite que le matériel ! Il tentait d'injecter l'adresse IP fixe alors que la carte Wi-Fi n'avait pas encore fini de s'initialiser au démarrage de Windows.

❌ Avant mon intervention
  1. Allumage du PC
  2. Lancement du script
  3. Tentative de configuration IP
  4. ÉCHEC — la carte Wi-Fi dort encore
✓ Après ma solution — boucle d'attente
  1. Allumage du PC
  2. Lancement du script
  3. Attente active de la carte Wi-Fi (boucle)
  4. Carte active ?
  5. Configuration IP
  6. SUCCÈS

Désormais, le script attend que l'interface soit bien active avant d'appliquer les paramètres.

✅ Déploiement fiable à 100 %, en seulement 3 à 5 minutes par machine.
Projet 2

🖥️ Affichage dynamique sous Linux avec Xibo

Mon second projet consistait à déployer et configurer une solution d'affichage dynamique pour la communication interne du site, en installant le lecteur Xibo Player sur un environnement Ubuntu 24.04.

Ce projet m'a permis de me frotter aux contraintes réseaux d'une grande entreprise : il a fallu configurer les variables d'environnement pour traverser proprement le proxy de la SCA Ouest et créer des scripts de démarrage automatique pour que l'affichage se lance tout seul dès le boot.
